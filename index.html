Команды Git
Команда Описание
git init Создание нового локального репозитория Git
git clone url Создание локальной копии удаленного репозитория Git
git add file Добавление конкретного файла в рабочей директории в индекс
git add . Добавление всех файлов в рабочей директории в индекс
git commit -m "comment" Для сохранения изменений, сделанных в локальном репозитории, в истории коммитов с комментарием
git commit -a -m "comment" Для сохранения изменений, сделанных в локальном репозитории, в истории коммитов с комментарием и автоматической добавлением все измененных файлов в индекс
git push url Для отправки изменений из локального репозитория в удаленный репозиторий Git
git status Для получения информации о текущем состоянии рабочего пространства Git
git log Для просмотра истории коммитов в репозитории Git
git branch branch_name Для просмотра, создания и удаления веток в репозитории Git
git checkout branch_name Для переключения между ветками, проверки коммитов и отката изменений
git checkout -b branch_name -
git remote -
